# Team Photo Naming Convention

Photos must be JPG format, named `firstname-lastname.jpg` using lowercase letters and hyphens only.

## Rules

- Lowercase everything
- Replace spaces with hyphens
- Drop titles (Dr., Mr., etc.)
- No special characters

## Examples

| Name | File |
|------|------|
| Kyle Matthys | `kyle-matthys.jpg` |
| Nawaf Alshaikhmubarak | `nawaf-alshaikhmubarak.jpg` |
| Dr. Purnima Voria | `purnima-voria.jpg` |
| Patience Nnedinso Eze | `patience-nnedinso-eze.jpg` |
| Parker Johnson | `parker-johnson.jpg` |

## Missing photos

If a photo file is not found, the component automatically shows the person's initials
on a green (#1A6B3C) background. No placeholder image needed.

## Recommended specs

- Square crop, face centered
- Minimum 200×200px (displayed at 32–96px)
- Max file size: 100KB per photo
